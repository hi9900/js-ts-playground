# 🚀 프로젝트 학습

JavaScript와 TypeScript로 다양한 프로젝트를 구현하는 폴더입니다.

## 📁 폴더 구조

- `games/` - 게임
- `ui-components/` - 재사용 가능한 UI 컴포넌트
- `utilities/` - 유틸리티 도구들
- `web-apps/` - 웹 애플리케이션
