# 🎵 아코디언 메뉴 (Accordion Menu)

접고 펼칠 수 있는 인터랙티브한 아코디언 메뉴 컴포넌트입니다.

## 기술 스택
- HTML5
- CSS3
- JavaScript

## 🚀 주요 기능
- 아코디언 패널 클릭 시 열기/닫기
- 한 번에 하나의 패널만 열리도록 제한
- 부드러운 슬라이드 애니메이션
- 반응형 디자인
