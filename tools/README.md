## 🚀 사용 방법

### 초기 설정

```bash
# 의존성 설치
npm install

# TypeScript 설정 파일 복사
cp tools/configs/tsconfig.json ./
cp tools/configs/.eslintrc.js ./
cp tools/configs/.prettierrc ./
cp tools/configs/jest.config.js ./
```

### 개발 워크플로우

```bash
# 코드 작성
npm run dev

# 코드 검사
npm run lint
npm run format

# 테스트 실행
npm run test

# 빌드
npm run build
```
