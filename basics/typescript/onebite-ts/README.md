## 🛠️ 개발 환경

### 필수 도구

- Node.js (LTS 버전)
- VS Code
- TypeScript Compiler

### 설치 명령어

```bash
# TypeScript 전역 설치
npm install -g typescript

# 프로젝트 초기화
npm init -y
npm install -D @types/node tsx

# TypeScript 설정 파일 생성
tsc --init
```
